XOR-Files
========
### Powershell &amp; Python XOR 2 Files

## Usage:
#### Python
```python
./xor.py file1 file2 outputFile
```

#### Powershell
```powershell
./xor.ps1 file1 file2 outputFile
```
## Example
#### Python
```
./xor.py C:\a.txt C:\b.txt c:\result.txt
```

#### Powershell
```
./xor.ps1 C:\a.txt C:\b.txt c:\result.txt
```
